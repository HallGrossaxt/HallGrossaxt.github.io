PICLas — unofficial Windows port

github.com/<handle>/piclas-windows

An unofficial Windows build of the open-source PICLas framework — a parallel particle-in-cell / DSMC code for simulation of rarefied non-equilibrium plasma flows. The upstream project targets Linux / HPC; this port aims to make local exploration on Windows workstations practical for students and engineers.
